C#

C# is an object-oriented programming language
from Microsoft that aims to combine the
computing power of C++ with the programming
ease of Visual Basic. C# is based on C++
and contains features similar to those
of Java.

Tips & Tricks

1. Add Access-Modifiers to your Properties in order to keep your Code clean and ordered.

2. Avoid writing the same line(s) of Code repeatedly - Maybe there is some Code you should tidy up a bit!

3. If you haven't yet - Learn Polymorphism! There's a link on Top if you wanna check it out!

4. Use the Advantage of beeing able to create your own Types of Objects! This is really helpful!
